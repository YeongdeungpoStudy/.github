# 영등포구 코테 스터디 👋

## 🔥 스터디 목표
- 잔디 채우기 🍀
- 취업 / 이직 코딩테스트 대비 📒

## ✏️ 스터디 진행 방식
- 매주 수요일 8시 영등포구청역 부근에서 코드 리뷰 및 풀이 설명하기 (약 2시간 소요)
- 일주일 단위로 알고리즘 3문제 화요일 오전까지 풀고 깃허브에 push
  - 백준 2문제(유형 선택) + 프로그래머스 1문제(랜덤 유형)

- 보증금 2만원 🤑
